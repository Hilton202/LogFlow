# 🚀 LogFlow System - Gestão Logística Inteligente

O **LogFlow** é um sistema de gestão de estoque desenvolvido para resolver problemas reais de logística, como a entrada manual de notas fiscais e a falta de rastreabilidade.

## ✨ Funcionalidades
- **Importação de NF-e (XML):** Leitura automática de produtos, lotes e preços direto do XML da nota fiscal.
- **Leitura de Código de Barras:** Integração com a câmera do dispositivo para identificação de produtos via código EAN.
- **Estoque em Tempo Real:** Banco de dados NoSQL integrado com Firebase Firestore.
- **Visão Financeira:** Registro histórico de entradas e custos unitários.

## 🛠️ Tecnologias Utilizadas
- **JavaScript (ES6+)**
- **Firebase** (Firestore & Hosting)
- **Quagga2** (Leitura de código de barras)
- **CSS3 / HTML5**
